## 🧑‍🤝‍🧑 팀원 구성
- 공통: DB설계 및 연동, 프론트 개발

- 김하경(팀장): 회원, 일류 로그인/회원가입, 관리자 페이지, 일류 검색, 일류 매칭

- 한유정(부팀장): 일류 프로필, 공통 MY프로필, 카테고리 관리, 프로필 작성

- 지은영(팀원): 커뮤니티, 게시글 신고/관리, 리뷰 작성, 일류 프로필

- 여의동(팀원): WebSocket 실시간 채팅, 실시간 알림

- 문성현(팀원): 게시글 관리, 포트폴리오 작성
